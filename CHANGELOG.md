## [0.1.0] - 2026-06-18

### Added
- Поиск файлов и папок по сетевым (SMB) путям через локальный индекс SQLite.
- Режимы поиска: подстрока, маска (`*?`) и регулярные выражения.
- Фильтры по размеру, дате изменения, типу/расширениям; «только файлы»/«только папки».
- Быстрые кнопки типов: PDF, Word, Excel, Фото.
- Поиск по содержимому в текущей выборке.
- Авто-обновление индекса по интервалу.
- Портативная публикация в один self-contained `.exe`.
- Контекстное меню строки: открыть файл, открыть папку, скопировать путь.
